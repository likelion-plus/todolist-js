## 구동

- JavaScript 프로젝트

```
cd 01.javascript
npx serve -s .
```

- TypeScript 프로젝트

```
cd 02.typescript/todoapp
npm run dev
```

- React 프로젝트

```
cd 03.react/todoapp
npm run dev
```

## [api 서버][] 구동

```
cd tododapi/api
npm i
npm start
```
